# Security Policy

## Supported Versions

The following versions of NGO Insights AI are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.0.1   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of NGO Insights AI seriously. If you believe you have found a security vulnerability, please do NOT open a public issue. Instead, please report it to us privately.

Please email security@ngoinsights.ai with details of the vulnerability. We aim to respond within 48 hours to acknowledge the report and provide an estimated timeline for a fix.
